Cisco交换机常用命令

发表于2024-05-16|更新于2024-05-16|路由交换
|阅读量:

用户模式
`# 特权模式
(config)# 全局模式
(config-if)# 接口配置模式
SVI(交换机虚拟接口)：为了远程管理访问交换机。需要给他配备默认网关。个人理解为VLAN接口。
给SVI VLAN99 分配地址172.17.99.11/24
configure terminal
interface vlan 99 //进入SVI(VLAN)的接口配置模式
ip address 172.17.99.11 255.255.255.0 给vlan99分配ip地址
no shutdown
ip default-gateway 172.17.99.1 //配置交换机的默认网关
验证ip设置：
show ip interface brief (全局模式下既可以使用)
//可以用来查看物理接口和虚拟接口的状态

配置交换机端口

configure terminal
interface f0/1 //进入接口配置模式
duplex full //配置接口双工模式
speed 100 //配置接口速率
end //返回特权模式
copy running-config startup-config
//将运行配置保存在启动配置中 这句命令是在特权模式下就可以执行

auto-MDIX 用来自动判断是直通线还是交叉线连接
使用auto-MDIX时必须将接口速率和双工模式都设置为auto,这样才能正常工作
启用auto-MDIX命令需要在交换机的接口配置模式下输入
S1(config-if)#mdix auto;
验证auto-MDIX设置:
show controllers ethernet-controller fa0/1 phy|include MDIX
//|为通道符号，与过滤器include （包含与过滤表达式相匹配的整段信息），section（以过滤表达式开始输出后面的控制信息），exclude（排除与过滤表达式相匹配的所有输出行），begin（从匹配筛选表达式的哪一行开始，显示所有输出行）

#交换机验证命令
show interfaces [inferface-id] //显示接口状态和配置
show startup-config //显示当前启动配置
show running-config //显示当前运行配置
show flash //显示有关闪存文件系统信息
show version //显示系统硬件和软件状态
show history //显示输入的命令历史记录
show ip interface //显示接口ip信息
show mac address-table //显示mac地址表

show interfaces常用来检测介质问题，输出的第一个参数(fastethernet0/18 is up)是指的硬件层，表示这个接口是否在接收载波检测信号，第二个参数(line protocol is up)指数据链路层。
interface:up line protocol:down 封装类型不匹配，另一端接口处在错误禁用(error-disabled)状态
interface:down line protocol:down 没连上，或接口另一端可能为管理性关闭状态(administratively down) //该状态可以通过no shutdown手动打开
show interfaces:可以检测输入错误，残帧，小巨人帧，CRC，输出粗我，冲突（半双工），延迟冲突

配置SSH

检查IOS是否支持SSH：show version ，版本名带k9就支持
配置过程：
1.验证是否支持SSH：show ip ssh
2.配置ip域名：ip domain-name name
3.生成RSA密匙对: crypto key generate rsa （删除为：crypto key zeroize rsa）
4.配置用户身份验证：本地身份验证的方法：username username secret password
5.配置VTY路线:
先用line vty range 进入配置vty的模式中
transport input ssh 在vty中启用SSH协议
login local 采用本地认证
6.启动SSH版本2
S1(config)ip ssh version 2

路由器的基本配置

配置路由器接口

1.配置至少一个ip地址 ip address ip-address subnet-mask
2.激活: no shutdown
3.描述:description ……可以为接口配置最多240字符的简短描述

ipv4环回接口

是一个虚拟的逻辑端口，只要路由器运行正常，该接口就会自动处于up状态
Router(config)# interface loopback number //启用环回接口
Router(config-if)# ip address 10.0.0.1 255.255.255.0 //将其配置ip
一个交换机可以不只有一个环回接口

验证直连网络

接口验证命令

show ip(ipv6) interface brief 显示所有接口的汇总信息，包括ip地址和接口的当前运行状态
show running-config interface interface-id:显示应用在指定接口上的命令
show ip route:显示存储在RAM中的ip路由表中的内容，活动接口应当出现在路由表中，而且具有两个相关条目，”C”(已连接)或”L”(本地)来标识

验证接口状态

show ip interface brief 可以查看所有接口的状态
show ip interface fasteither-name 查看某一个接口的状态

验证接口的配置

show running-config interface 显示应用到指定端口上的当前命令
show interfaces 显示所有接口的接口信息和数据包流量的计数
show ip interface 显示路由器所有接口与ip有关的信息

过滤show命令输出

terminal length 数量 可以生成单页更多行，0值可以防止路由器在输出屏幕之间暂停
|为通道符号，与过滤器include （包含与过滤表达式相匹配的整段信息），section（以过滤表达式开始输出后面的控制信息），exclude（排除与过滤表达式相匹配的所有输出行），begin（从匹配筛选表达式的哪一行开始，显示所有输出行）
